# Discord.js Timeout

<h2>Temel Fonksiyonlar</h2>

```js
const timeout = require('./index.js') //npm'e yükleyince 'npm i timeout.js' ile yükleyebilceksiniz
var token = "token"
var guild = "sunucu id"
var user = "mutelencek kişi id"
timeout.on(token, guild, user, "süre") //süre => 10m - 1h gibi
```
